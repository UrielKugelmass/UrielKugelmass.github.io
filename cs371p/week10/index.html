5. What's my pick-of-the-week or tip-of-the-week?
My tip of the week is a tool/shortcut for iterm. I reccommended iterm instead of the regular terminal (for MacOS users) a while ago. Today I want to reccommend hotkeys. You can customize whatever key commands you want and iterm will pop-up immediately. It is an easy and quick way to get the terminal running by pressing 1-2 keys instead of doing cmd-space + "iterm" + return. It also opens up faster and in a smooth way.
