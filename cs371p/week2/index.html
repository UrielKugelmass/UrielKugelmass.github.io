As far as content is concerned, I feel like I am learning, I enjoyed last lecture’s distinction between pre and post-fix increment for a variable/custom-defined-object.
5. What's my pick-of-the-week or tip-of-the-week?
My tip of the week is “View in browser” plugin. I use the version for sublime, but there is a visual studio code version as well and I am sure there are others for your favorite IDE also. This tool is super useful when working on front-end projects or even this class’s blog, as it shows you how your code would look on a browser without pushing it to source control. This makes debugging easier and the whole process takes less time. It is similar to gulp watch but easier to set up in my opinion.
